# NFT Marketplace — Test Task

### 🛠 Стек технологий

- **Core:** Next.js 16 (App Router), TypeScript.
- **State & API:** Redux Toolkit + RTK Query (асинхронные запросы, кэширование, трансформация DTO).
- **Architecture:** **Feature-Sliced Design (FSD)**
- **UI/UX:** SCSS Modules, Framer Motion (анимации), Keen-Slider, Skeleton Screens.
- **DevOps:** Docker (Multi-stage build)

---

### 🚀 Быстрый запуск

**Вариант 1: Docker (рекомендуется)**

```bash
docker-compose up -d --build
```

После запуска приложение доступно на: http://localhost:3000

**Вариант 2: Стандартный запуск**

```bash
npm install
```

```bash
npm run dev
```

### Сборка и запуск в production режиме

```bash
npm run build
```

```bash
npm run start
```
